Welcome to the official documentation for **Terax**, the terminal-first AI-native developer workspace.

Terax brings your terminal, text editor, AI agents, and live browser preview together into a single, cohesive, sub-millisecond environment.

### Core Philosophy

Terax was built to solve the fragmentation of modern development tools. Instead of context switching between a terminal app, a text editor, an AI helper, and a web browser, Terax embeds all four into one unified window:

- **Keyboard First:** Designed to be operated entirely from your keyboard with intuitive command shortcuts.
- **Agentic Coding:** Run powerful AI agents that can read project structures and propose file changes in reviewable diffs before writing to disk.
- **Zero Latency:** High-performance rendering pipeline using WebGL and Rust backend utilities.
