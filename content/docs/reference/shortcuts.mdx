Terax is designed to be keyboard-first. Most workspace actions can be triggered using shortcuts.

## Global Navigation

On macOS, use the `Cmd` key. On Linux and Windows, use the `Ctrl` key.

| Shortcut          | Command ID        | Action                                               |
| :---------------- | :---------------- | :--------------------------------------------------- |
| `Cmd + T`         | `tab.new`         | Open a new terminal tab                              |
| `Cmd + W`         | `tab.close`       | Close the active workspace tab                       |
| `Cmd + \`         | `sidebar.toggle`  | Toggle the sidebar panel (explorer / source control) |
| `Cmd + J`         | `ai.toggle`       | Toggle the docked AI Composer bar                    |
| `Cmd + P`         | `finder.files`    | Open the fuzzy file finder                           |
| `Cmd + Shift + P` | `command.palette` | Open the workspace command palette                   |

## Editor Shortcuts

These actions are active when an editor tab is focused.

| Shortcut  | Action                                          |
| :-------- | :---------------------------------------------- |
| `Cmd + F` | Open the inline search finder                   |
| `Cmd + /` | Toggle comment on active lines                  |
| `Tab`     | Accept AI autocomplete completion suggestion    |
| `Esc`     | Clear active selection / exit visual mode (Vim) |
