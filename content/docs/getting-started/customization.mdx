Terax is designed to be fully customized to match your development aesthetic. From editor color schemes to file explorer icons, everything is configurable.

## Themes & Presets

Terax comes prebuilt with several terminal and editor color palettes:

*   **Catppuccin (Mocha/Macchiato):** Soft, high-contrast pastel colors.
*   **Tokyo Night:** A dark, vibrant theme inspired by the lights of Tokyo.
*   **One Dark:** The classic, clean Atom-inspired syntax scheme.
*   **Dracula:** A classic dark, high-contrast palette.

To cycle through themes, use the command palette (`Cmd + Shift + P`) and type `Preferences: Select Color Theme`.

---

## Configuration File

Your preferences are stored in a simple JSON file located in your home directory:

*   **macOS / Linux:** `~/.config/terax/config.json`
*   **Windows:** `%USERPROFILE%\.config\terax\config.json`

Here is an example customization snippet:

```json
{
  "theme": "catppuccin-mocha",
  "font_family": "Geist Mono",
  "font_size": 13,
  "vim_mode": true,
  "ai": {
    "provider": "openai",
    "model": "gpt-4o",
    "autocomplete": true
  }
}
```

## Customizing the Shell

Since Terax launches your default system shell (`zsh`, `bash`, or `fish`), you can continue using your favorite prompts like **Starship** or **Oh My Zsh**. 

For optimal performance, make sure to enable the WebGL renderer in your configuration settings:

```json
{
  "terminal": {
    "renderer_type": "webgl"
  }
}
```
This ensures sub-millisecond keystroke latency and smooth scrolling even with heavy command logs.
