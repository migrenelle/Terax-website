Welcome to **Terax**, the terminal-first AI-native developer workspace. Terax brings your terminal, text editor, AI agents, and live browser preview together into a single, cohesive, sub-millisecond environment.

## Installation & First Run

After downloading and launching the application, you'll be greeted with a clean, hardware-accelerated command shell. 

### Key Shortcuts

To navigate the Terax workspace efficiently, use the built-in keyboard shortcuts:

| Command | Action |
| :--- | :--- |
| `Cmd + T` | Open a new terminal tab |
| `Cmd + W` | Close the current tab |
| `Cmd + \` | Toggle terminal sidebar (file explorer) |
| `Cmd + J` | Open the AI prompt input |
| `Cmd + P` | Fuzzy search files in the current project |

---

## Workspace Layout

Terax workspace consists of four main components designed to work side-by-side:

1. **Terminal:** Silky smooth WebGL-rendered `xterm.js` instances.
2. **Editor:** Vim-mode enabled text editor with context-aware AI completions.
3. **AI Panel:** Agent workflows that can read files, propose diffs, and execute commands.
4. **Browser Preview:** Automatic dev-server detection for hot-reloading your web projects.

```bash
# Verify your installation
terax --version
```
