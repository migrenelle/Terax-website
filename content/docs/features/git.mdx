Terax contains full source-control integration, featuring a git dashboard, side-by-side diff comparisons, and a visual commit history.

## Staging and Committing

The Source Control panel displays file modifications grouped by their status:

- **Unstaged Changes**: Modified files that are not staged. Hovering reveals actions to stage individual files or discard changes.
- **Staged Changes**: Files prepared for the next commit.
- **Commit Bar**: Type a commit message directly in the sidebar input field and commit changes.

Terax runs Git operations using a dedicated git helper module in the Rust backend. Operations are gated through the workspace authorization registry to verify permissions.

## Branch Diffs

Clicking a modified file opens a side-by-side visual diff. The diff highlights line insertions and deletions using standard color palettes derived from your active theme.

## Commit History Graph

The Git History panel displays a commit graph. Features include:

- **Commit Rail**: Visual branch intersections and merge lines.
- **References**: Local and remote branch pointers (like `HEAD`, `origin/main`).
- **Commit Diffs**: Select any historical commit to view its details and modified file diff lists.
